# 🌸 Sakura SSH Client

A beautiful, anime-themed desktop SSH client built with **Electron.js**. Featuring glassmorphism, Sakura-petal particles, and advanced SSH/SFTP capabilities.

![Anime SSH Client](https://images.unsplash.com/photo-1614850523296-62c0af475455?q=80&w=1470&auto=format&fit=crop) *(Replace with your screenshot!)*

## ✨ Features

- **🌸 Anime "Sakura Night" Theme**: Deep midnight-blue aesthetic with soft pink highlights.
- **🖼️ Custom Wallpaper**: Upload your own background image in the Settings.
- **🛡️ Encrypted Credentials**: Secure local storage with AES-256 encryption.
- **💻 Multi-Tab Terminal**: Manage multiple sessions at once with ease.
- **📁 Integrated SFTP**: Seamlessly browse, download, and upload files.
- **🗝️ SSH Key Support**: Authenticate using private keys (PEM, OpenSSH) with passphrase protection.
- **🖥️ Cross-Platform**: Runs on Windows, macOS, and Linux.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Version 16 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/Nonx2360/SSH-Client.git
   cd SSH-Client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## 🛠️ Built With
- **[Electron](https://www.electronjs.org/)** - Desktop application framework
- **[SSH2](https://github.com/mscdex/ssh2)** - SSH and SFTP library for Node.js
- **[xterm.js](https://xtermjs.org/)** - Terminal front-end component
- **[electron-store](https://github.com/sindresorhus/electron-store)** - Encrypted local storage for profile management

## 🎨 Theme Customization
You can change the **Accent Color**, **Font Size**, and **Wallpaper** directly within the application's built-in settings panel (the gear icon on the sidebar).
